# 展望

- 时间序列的预测ARIMA和LSTM都可以，前者主要针对线性问题，后者在非线性问题上表现不错，看到有的论文把ARIMA和LSTM“串联”起来，效果好像不错。没时间了，回头试试。
- 因为有节假日，所以数据会有局部的突然增大和减小，但是预测的15天是没有节假日的。可以根据百分位数或聚类等方法识别异常点，然后用所有数据里面每周对应这天的平均值替换掉。
